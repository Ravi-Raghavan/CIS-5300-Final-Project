# CIS 5300: Term Project

This repository contains the code, data, and outputs for the CIS 5300 Term Project.

## Directory Structure

- [`code/`](code/README.md): Contains all scripts and notebooks used for training models, preprocessing data, and generating outputs.
- [`data/`](data/README.md): Contains all datasets used for the project, including raw data and any processed versions.
- [`output/`](output/README.md): Contains model outputs such as predictions, evaluation metrics, and any generated results.
